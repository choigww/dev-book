1. 연산의 중첩에서 연산의 '임시결과'란 무엇을 의미하며, 임시결과의 가장 중요한 특징은 무엇인지 기술하세요.

> 연산이 여러개 진행되는 과정에서, 최종 연산까지 도달하기 이전 연산 결과를 '임시결과'라고 부른다.
> 임시결과의 가장 중요한 특징은, 가장 최근 임시결과 하나만을 저장하며, 별도 변수에 저장하지 않으면 사라진다는 것이다.


2. char형 자료와 int형 자료 간에 +연산을 수행하면 임시결과의 자료형은 무엇인지 답하세요.

> int형 자료


3. 2번 문제와 같은 현상을 부르는 용어는 무엇이며, 이형자료 간의 연산에서 임시결과에 대한 자료형이 결정되는 근거는 무엇인지 기술하세요.

> 형승격(Type Promotion)
> 이형자료 간의 연산에 문법적으로 문제가 없다면, 표현범위가 더 넓은 자료형으로 통일된다.


4. 5 / 2 연산 결과의 자료형식과 값은?

> int, 2


5. 사용자로부터 정수를 입력받아 5로 나눈 나머지를 출력하는 프로그램을 작성하세요.

#include <stdio.h>
int main(void)
{
	int nInteger = 0;
	printf("정수를 입력하세요: ");
	scanf("%d", &nInteger);
	printf("5로 나눈 나머지는 %d입니다.\n", nInteger % 5)
	return 0;
}


6. 대입 연산자의 왼쪽 피연산자가 될 수 있는 자격을 갖춘 대상을 부르는 용어는?

> l-value


7. 다음 코드의 실행결과는 무엇인지 쓰세요.

#include <stdio.h>
int main(void)
{
	printf("%08X\n", 0xFFFFFFFF & 0x00080000);
	printf("%08X\n", 0XAAAABBBB & 0x00080000);
	printf("%08X\n", 0xAABBCCDD & 0x00080000);

	printf("%08X\n", 0xFFFFFFFF | 0x00080000);
	printf("%08X\n", 0x11223344 | 0x00080000);
	printf("%08X\n", 0x00000000 | 0x000F0000);

	return 0;
}

> 00080000
> 00080000
> FFFFFFFF
> 112A3344
