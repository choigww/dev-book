## Getting Started

- 하루 30분에서 1시간을 할애하여 책을 읽습니다.
- 개발과 직접적 관련이 높은 책을 먼저 읽습니다.
- 직접적 관련은 없으나 `근본`이 된다면 먼저 읽습니다.
- 독서도 좋지만, 나의 CORE FUNCTION이 무엇인지 잊지 않습니다.
- `지금`의 상황에서 유연하게 계획을 조정합니다.
